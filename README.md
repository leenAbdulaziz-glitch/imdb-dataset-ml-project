# imdb-dataset-ml-project
# IMDB Movie Analysis & Prediction

This project analyzes IMDb movie data and builds machine learning models to predict movie ratings.

Contents:
- Data analysis and visualization
- Machine learning models
- Streamlit application

Files:
imdb_app.py → Streamlit web application  
IMDb_AllInOne2.ipynb → Full analysis notebook  
Datasets → imdb_top_1000.csv & imdb_clean.csv  
Visualizations → PNG charts
